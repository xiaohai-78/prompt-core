“文本转 JSON” 会解析当前文本并以易读格式展示。
“JSON 压缩文本” 会去掉多余空格和换行，适合传输或存储。
“转义还原” 可把 \n、\"、\/、\\ 等还原成原始文本。
“MD 预览” 会把当前文本按 Markdown 格式渲染展示。
首次点“绑定历史目录”后，可把历史自动同步到同级目录中的 json-tool-history.jsonl。
历史记录保存在浏览器本地，不上传网络；页面仍是单 HTML 文件，直接双击即可使用。
